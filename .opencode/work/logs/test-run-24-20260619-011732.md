Full workflow structure:
```
name: E2E Tests
on:
  push (branches: [main], paths-ignore)
  schedule (cron: 0 6 * * *)
permissions: contents:read, issues:write
jobs:
  e2e:
    runs-on: ubuntu-latest
    timeout-minutes: 15
    env: CI=true
    steps:
      1. actions/checkout@v4
      2. docker/setup-buildx-action@v3
      3. Start services (docker compose up -d --build)
      4. Wait for backend health (curl loop, 30 attempts)
      5. actions/setup-node@v4 (node 20)
      6. Install Playwright (npm ci + npx playwright install chromium)
      7. Run Playwright tests (npx playwright test)
      8. Upload report (upload-artifact@v4, always)
      9. Create issue on failure (github-script@v7, main only)
```
