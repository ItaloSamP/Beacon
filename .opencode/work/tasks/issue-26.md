# Task: issue-26 — Quality Hub HTML Dashboard on GitHub Pages

## Status: DONE

## Metadata
- **Type:** feature
- **Scope:** frontend
- **Priority:** high
- **Source:** GitHub Issue #26

## Problem Statement
Create a beautiful, interactive HTML dashboard hosted on GitHub Pages that reads `reports.json` (generated by the reports aggregator #25) and displays all quality metrics in one place. Uses Beacon's design system (TailwindCSS, Inter font, Lucide icons). No build step — pure HTML/CSS/JS.

## Acceptance Criteria
- [ ] 5 status cards (Backend / Frontend / Agent / Docker / E2E) with pass/fail badge, test counts, coverage %
- [ ] Coverage trend section with mini sparkline charts
- [ ] Security section: Bandit issues grouped by severity (HIGH/MEDIUM/LOW)
- [ ] Latest test runs table with pass/fail counts and links to CI runs
- [ ] Lint warnings table grouped by file
- [ ] Docker images section with sizes and tags
- [ ] History table: last 20 runs with date, commit, status, duration
- [ ] Filter by component (Backend/Frontend/Agent/Docker/E2E)
- [ ] Filter by date range
- [ ] Drill-down: click a test failure → expand stack trace
- [ ] Drill-down: click a security issue → expand CWE, file, line, description
- [ ] Dark mode toggle (respects prefers-color-scheme)
- [ ] Auto-refresh every 60s
- [ ] Export: download reports.json or copy table as CSV
- [ ] GitHub Pages deploy workflow to `/hub/` directory

## Technical Approach
**Decision:** Pure HTML/CSS/JS static dashboard with CDN dependencies
**Origin:** user-driven (from issue spec)
**Rationale:** GitHub Pages serves static files — no build step needed. TailwindCSS, Lucide, Inter via CDN keeps dependencies light. Vanilla JS is sufficient — page complexity doesn't justify React.

## Architecture Fit
- Standalone — does not touch existing Beacon codebase
- Deploys to gh-pages branch under `/hub/` directory
- Coordinates with #31: Doc Hub deploys to root `/`, Quality Hub to `/hub/` — path isolation prevents overwrites
- Design system continuity: same TailwindCSS tokens, Inter font, Lucide icons as Beacon frontend
